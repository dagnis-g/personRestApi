# personRestApi

REST API using Java, Spring Boot, H2 and Angular

### How to run

To run Spring Boot application. In project root directory

```shell
mvn spring-boot:run
```

To run Angular navigate to src/main/resources/frontend

```shell
npm install 
ng serve

Browser: http://localhost:4200/
```
